Q. API란 무엇인가?

```
API는 일종의 계약서로 생각할 수 있습니다. 개발자는 외부 서비스나 시스템의 기능을 활용하기 위해
API를 사용하여 요청을 보내고, 응답을 받아 처리합니다. 이를 통해 다른 개발자나 서비스가 제공하는 기능을 직접 호출하여 사용할 수 있습니다.

API는 다양한 형태로 제공될 수 있는데, 웹 API의 경우 HTTP를 통해 요청과 응답을 주고받는 RESTful API가 흔히 사용되며, JSON 또는 XML 형식으로 데이터를 주고 받습니다.

정리하자면 API는 개발자에게 다른 시스템의 기능을 쉽게 활용할 수 있는 기회를 제공하며, 시스템 간의 기능 확장에 유용합니다.

다양한 서비스와 플랫폼이 API를 제공하고 있으며, 개발자는 이를 활용하여 자신의 애플리케이션을 더욱 풍부하게 만들 수 있습니다.
```
