Q. HTTP와 HTTPS에 대해 설명해주세요.

```
HTTP는 인터넷에서 하이퍼 텍스트를 교환하기 위한 통신규약입니다. 사용자가 브라우저에 방문을 하면 브라우저가 웹 서버에 HTTP 요청을 전송하고 웹 서버는 HTTP 응답으로 응답합니다. 이때 웹 서버와 사용자 브라우저는 데이터를 일반 텍스트로 교환하는데, 이를 HTTP라고 합니다. 하지만 HTTP의 경우 암호화가 되지 않은 데이터를 전송하는 프로토콜이기 때문에 중요정보를 주고 받으면 제3자가 정보를 조회할 수 있어 안전하지 않습니다. 이러한 문제를 해결하기 위해 HTTPS가 등장하게 되었는데 HTTPS는 SSL 또는 TSL프로토콜을 사용하여 통신을 암호화 하기 때문에 사용자의 개인정보와 데이터를 보호할 수 있습니다. 하지만 HTTPS는 SSL 또는 TSL과 같은 인증서를 발급하고 유지하기 위한 추가비용이 발생한다는 단점이 있지만, 웹 보안과 사용자 신뢰를 확보하기 위해서는 이러한 추가 비용을 감수하는 것이 일반적으로 권장됩니다.
```
