Q. 스코프에 대해 알고있는가?

```
js에서 스코프는 변수와 함수의 유효 범위를 정의하는 개념입니다.
js에서는 세가지의 스코프 종류가 있는데 전역 스코프, 함수 스코프, 블록 스코프가 존재합니다.

먼저 전역 스코프의 경우 코드의 가장 바깥쪽에 정의되는 스코프로, 어느 곳에서도 접근 가능한 범위입니다.
전역 스코프에서 선언된 변수와 함수는 어디서든지 접근이 가능하고 사용이 가능합니다.

함수 스코프의 경우 함수 내부에서 정의된 변수와 함수가 해당 함수 내부에서만 접근 가능한 범위입니다.
함수 내부에서 선언된 변수와 함수는 외부에서는 접근이 불가능합니다.

블록 스코프는 ES6부터 도입이 되었으며 중괄호로 둘러싸인 코드블록 내에서 정의된 변수가 해당 블록 내에서만 접근 가능한 범위입니다.
예를 들어 if문이나 for문 등의 블룩 내에서 선언된 변수는 블록 내부에서만 접근이 가능하고 외부에서는 접근할 수 없습니다.

스코프는 변수를 찾을 때 가장 가까운 스코프부터 시작하여 상위 스코프로 타고올라가면서 변수를 찾는 스코프체인을 하게됩니다.
이러한 스코프 체인을 통해 변수의 유효 범위를 확인하고 변수의 충돌을 방지합니다.

또한 스코프는 변수의 유효 범위를 제한하므로 충돌을 방지할 수 있게 해주고 코드의 안정성을 높여줍니다.
스코프를 이해하고 적절하게 활용하는 것은 js에서 중요한 부분이라고 할 수 있습니다.
```
