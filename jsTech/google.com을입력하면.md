Q. 브라우저 주소창에 google.com을 입력하면 무슨일이 발생하는가?

```
DNS 조회: 브라우저는 입력한 주소인 "google.com"을 DNS(Domain Name System)에게 전달하여 해당 도메인의 IP 주소를 조회합니다. DNS는 도메인 이름을 IP 주소로 변환해주는 역할을 합니다.

서버 연결: DNS 조회를 통해 얻은 IP 주소로 서버와의 연결을 시도합니다. 브라우저는 해당 IP 주소로 HTTP 요청을 보냅니다.

HTTP 요청: 브라우저는 HTTP 프로토콜을 사용하여 서버에게 요청을 전송합니다. 요청에는 HTTP 메서드(GET)와 요청한 리소스의 경로("/") 등이 포함됩니다.

서버 응답: 서버는 브라우저의 요청을 받고 처리한 후, 응답을 생성하여 브라우저로 전송합니다. 응답은 일반적으로 HTML 문서와 함께 HTTP 상태 코드(200 OK) 등의 정보를 포함합니다.

HTML 렌더링: 브라우저는 서버로부터 받은 HTML 문서를 파싱하여 DOM(Document Object Model) 트리를 구성합니다. 이후 CSS 파일, JavaScript 파일 등의 외부 리소스를 다운로드하고 처리합니다.

페이지 표시: 브라우저는 DOM 트리와 CSS 스타일 정보를 사용하여 페이지를 렌더링하고 화면에 표시합니다. 사용자는 구글 검색 엔진의 홈페이지를 볼 수 있게 됩니다.

이러한 과정을 통해 브라우저는 사용자가 입력한 주소로부터 해당 웹사이트의 컨텐츠를 가져와 화면에 표시합니다.
```
