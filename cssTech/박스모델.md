Q. 박스모델에 대해 설명해보세요

```
모든 HTML 태그는 박스 모양으로 구성되고 이를 박스 모델이라 부릅니다.
이 박스모델은 Content, Padding, Border, Margin으로 구성되는데

Content는 요소의 텍스트나 이미지 등 실제 내용이 위치하는 영역이며 width, height 프로퍼티를 갖습니다.
Padding는 Border 안쪽에 위치하는 요소의 내부 영역입니다.
Border는 테두리 영역입니다.
Maring은 테두리 바깥에 위치하는 요소의 외부 영역을 의미합니다.

또한 CSS에는 크게 두가지 박스의 유형으로 나뉘는데 블록 박스와 인라인 박스로 나뉘게 됩니다.

블록 박스의 대표적인 태그는 div태그로 특징으로는 width와 height값이 적용 되며 해당 영역을 모두 차지합니다.
인라인 박스의 대표적인 태그는 spanㅌ 태그로 블록박스와는 반대로 width와 height 값의 적용이 불가능하며 content영역 만큼만 영역을 차지하게 됩니다.

따라스 인라인 박스에 width나 height를 적용시켜주고 싶다면 display를 inline-block으로 바꾸어주게 되면 블록 박스와 마찬가지로
높이와 너비를 설정해줄 수 있게 됩니다.
```
