Q. React의 JSX에 대해 설명해보세요.

```
JSX는 JavaScript 확장 문법입니다.
JSX를 사용하면 Js코드에서 HTML과 유사한 구조와 태그를 사용하여 컴포넌트의 구조와 모습을 표현할 수 있습니다.
또한 중괄호로 감싸사 JS 표현식을 자유롭게 사용이 가능하며 조건부 렌더링을 해줄수도 있습니다.
이로인해 React 컴포넌트의 가독성을 높이고 작성하기 쉽게 만들 수 있습니다.
React에서 이러한 JSX문법을 자유롭게 사용가능한 이유는 Babe,l과 같은 트랜스 파일러가 JSX 코드를 JS코드로 변환해주기 때문입니다.
```

Q. 그렇다면 브라우저가 JSX 파일을 읽을 수 있는가?

```
브라우저는 JSX 파일을 직접 읽을 수는 없습니다. 브라우저가 JSX 파일을 읽으려면 JSX를 자바스크립트 객체로 변환을 해야합니다.
그리고 그 변환 작업은 바벨과 같은 컴파일러를 통해 이루어집니다.
따라서 바벨과 같은 컴파일러 덕분에 JSX 파일을 자바스크립트 객체로 변환해주고 브라우저는 변환된 자바스크립트 객체를 읽을 수 있게 됩니다.
```
